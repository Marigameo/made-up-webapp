## UI challenge - e-commerce web app

* React.JS

## Steps to clone and run the application

* git clone https://github.com/Marigameo/made-up-webapp.git

* yarn install 

* yarn start

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
